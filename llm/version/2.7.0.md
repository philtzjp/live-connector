# 2.7.0

ノートをサーバー側で変換する `transform_notes` ツールを追加する minor release。

## 追加機能

- `transform_notes`: select で選んだ 1 つの MidiClip のノートを決定的に変換する。全ノートを MCP クライアントへ往復させずに済む。
  - `transform`: `transpose{semitones}` / `time_shift{delta}` / `velocity{scale?,offset?}` / `quantize{grid,strength?}` / `duplicate{offset,count?}`。
  - `filter`: `timeStart` / `timeEnd`（クリップ相対拍）と `pitchMin` / `pitchMax` で対象ノートを絞る。フィルタ外のノートは素通し。
  - 座標はクリップ相対拍で完結。結果の `startTime` が `[0, クリップ長)` を外れる場合は `onOutOfRange`（`drop` 既定 / `error`）で扱う。`pitch` が `0..127` を外れるノートは削除する。
  - `preview` で変換前後のサマリ（対象ノート数・pitch/時間の範囲・削除数）を返す。

## 変更内容

- 純粋関数 `transformNotes` を抽出し、変換ロジックを SDK 非依存で固定する。
- `query_contract.select.tools` と `llm/models.yaml` に `transform_notes` を追加する。

## 動作確認

- `pnpm test`: transpose（pitch 範囲外削除）/ time_shift（範囲外 drop・error）/ velocity（clamp）/ quantize（グリッド吸着）/ duplicate（複製・範囲外 drop）/ filter（対象限定）を検証。
- `pnpm typecheck`: 全パッケージ成功。

## 制約

- ランダム性を伴う humanize 等は本 release の対象外（決定的変換のみ）。
- `velocity` 変換は未設定ノートの velocity を既定 100 として扱う。
