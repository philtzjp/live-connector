# 2.17.9

transform_notes と batch を書き込み前スナップショットの対象に加える patch release。

## 修正内容

- `transform_notes` が適用直前に旧 notes をスナップショットし、応答に `snapshotId` を返す。restore_snapshot で変換前へ巻き戻せる。
- `batch` が各ステップの適用直前（confirm 通過後）にスナップショットを取得し、応答の `snapshots: [{index, tool, snapshotId}]` で返す（ok と failed/apply の両方）。write_notes ステップは notes スナップショット、set_* ステップは properties スナップショット（tool は `batch:<tool>` として記録）。同一対象への多段ステップでは先頭ステップの snapshotId が batch 実行前の状態を表す。
- SERVER_INSTRUCTIONS の snapshotId 案内を transform_notes / batch を含む記述に更新する。

## 動作確認

- `pnpm test`: transform_notes の ok 応答に snapshotId が含まれ restore_snapshot で変換前へ戻ること、batch の snapshots から restore で batch 前へ戻ることを検証（extension 84 件全緑）。
- `pnpm typecheck`: 全パッケージ成功。
