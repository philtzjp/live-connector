# 2.9.0

サンプルを Simpler へ読み込む `load_sample` ツールを追加する minor release。

## 追加機能

- `load_sample`: select で選んだ 1 つの Simpler に、`audioFilePath` のオーディオファイルを読み込む。`Resources.importIntoProject` でプロジェクトフォルダへ取り込み、`Simpler.replaceSample` で適用する。取り込み後の管理パスを `importedPath` として返す。
- Drum Rack のパッド（`DrumChain` 配下の Simpler）も `HAS_CHAIN` → `HAS_DEVICE` で辿れるため対象に含まれる。
- Simpler 未挿入のトラックを対象にした場合は、`insert_device` で Simpler を挿入する導線を hint で案内する。

## 変更内容

- 入力検証: `audioFilePath` の絶対パス・対応拡張子（`.wav` / `.aif` / `.aiff` / `.mp3` / `.flac` / `.ogg` / `.m4a` / `.aac`）と、ファイル存在（`fs.stat`）を検証し、失敗時は hint 付き構造化エラー（不存在は `not_found`）を返す。
- `importIntoProject`（ファイルコピー）はトランザクション外、`replaceSample`（デバイス変更）は `withinTransaction` 内で実行する。
- `query_contract.select.tools` / `llm/models.yaml` / `llm/ARCHITECTURE.md` を更新する。

## 動作確認

- `pnpm test`: `isSupportedAudioPath` / `assertSamplePath`（絶対パス・形式検証）を検証。
- `pnpm typecheck`: 全パッケージ成功。

## 制約

- `search_presets` で列挙したオーディオファイルのパスをそのまま `audioFilePath` に使用できる。`.adv` / `.adg` などのプリセットファイルは対象外（Simpler が読むのはオーディオのみ）。
