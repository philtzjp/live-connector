# 2.1.0

Track のデバイスチェーンへ内蔵 Live デバイスを挿入する `insert_device` ツールを追加する minor release。MIDI トラックにインストゥルメントを挿入してノートを発音可能にする。

## 追加機能

- `insert_device` ツールを追加する。`select` で Track を 1 つ特定し、内蔵 Live デバイス名と挿入 index を受け取り `Track.insertDevice(deviceName, index)` を呼ぶ。index 省略時はデバイスチェーンの末尾に追加する。
- 挿入失敗時は内蔵デバイス名カタログを `validDeviceNames`（`McpErrorMetadata` / `McpError` の新フィールド）として返す。Ableton Extensions SDK v1.0.0-beta.0 にデバイス列挙 API（Browser）が無いため、Browser ツリー探索／列挙の代替とする。
- `insert_device` は `preview`（ドライラン）に対応する。

## 修正内容

- 内蔵デバイス名カタログ（`BUILT_IN_DEVICE_CATALOG`）から、実機で `failed to insert` となる `Bass` を除外する。

## 動作確認

- `insert_device` で MidiTrack に `Operator` / `Electric` を挿入し、`query` の `HAS_DEVICE` に反映され、ノートが発音すること（2026-06-28 実機検証済み）。
- `/health` が `application/health+json` で `version: "2.1.0"` を返すこと。
- 配布アーカイブは `pnpm package` で `apps/extension/dist/live-connector-2.1.0.ablx` に生成すること。

## 制約

- `insertDevice` は内蔵 Live デバイスのデフォルトプリセットのみ挿入する。サードパーティプラグインおよび `.adv` / `.adg` / `.vstpreset` / `.aupreset` のプリセットファイルロードは SDK 非対応。
- カタログは網羅的ではなく、未掲載・名称不一致のデバイス名は `validDeviceNames` ヒント付きでグレースフルに失敗する。正確な名称は Live のブラウザ表示に合わせること。
- サンプルロード（`Simpler.replaceSample`）はデバイス挿入とは別関心のため本リリースに含まない。
