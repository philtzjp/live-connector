# 2.17.1

主要ツールの入力スキーマ・説明・エラー文言を補完し、試行錯誤を不要にする patch release。

## 修正内容

- `create_arrangement_clip`: `startTime`（アレンジメント絶対拍）/ `duration`（クリップ長・拍。`length` ではない）/ `name` / `audioFilePath`（AudioTrack で必須）/ `isWarped` に `describe` を追加。説明に必須項目と最小例を明記する。
- `write_notes`: ノート各要素（`pitch` / `startTime`（クリップ相対拍）/ `duration` / `velocity` ほか）に単位付き `describe` を追加。`notes` フィールドに要素の形と最小例（`[{pitch:60,startTime:0,duration:1,velocity:100}]`）を明記する。
- `set_*` の誤選択・非存在エラーを実用的にする。ラベル不一致は「`<tool>` edits existing `<Label>` nodes, but the select matched a `<Other>` node」、0 件は「select matched no `<Label>` to edit」とし、編集専用ツール（`set_track` / `set_scene` / `set_cue_point`）には新規作成ツール（`create_track` / `create_scene` / `create_cue_point`）を案内する `createHint` をエラーに付ける。
- `set_track` / `set_scene` / `set_cue_point` の説明を「編集専用（既存対象）」＋新規作成ツールの案内に更新。`set_track` にミキサー Parameter 経路（`set_device_parameter`）を注記。

## 動作確認

- `pnpm test`: `set_cue_point` が非 CuePoint を選択した場合・該当 CuePoint が無い場合に、`create_cue_point` を案内するエラーを返すことを検証。
- `pnpm typecheck`: 全パッケージ成功。既存の正常系挙動は不変。
