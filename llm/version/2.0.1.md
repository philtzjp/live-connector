# 2.0.1

Ableton Extension Host の VM コンテキストで `AbortSignal` などの Web 標準 globals が未定義になる問題を修正する patch release。

## 修正内容

- Extension Host の外側の Node ランタイムに存在する Web 標準 globals を、バンドル先頭で VM コンテキストへコピーする。
- `AbortSignal` / `AbortController` / `Request` / `Response` / `Headers` / `fetch` などを、SDK と MCP HTTP transport の読み込み前に初期化する。
- `.ablx` 配布物の manifest、HTTP health response、MCP server metadata のバージョンを `2.0.1` に更新する。

## 動作確認

- Ableton Live の Extension Host が `Failed to load extension: AbortSignal is not defined` で停止しないこと。
- `/health` が `application/health+json` で `version: "2.0.1"` を返すこと。
- 配布アーカイブは `pnpm package` で `apps/extension/dist/live-connector-2.0.1.ablx` に生成する。
