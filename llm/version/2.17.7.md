# 2.17.7

transform_notes の範囲外ノート処理を onOutOfRange 指定に従わせる patch release。

## 修正内容

- pitch が 0..127 を外れたノートが `onOutOfRange:"error"` 指定時でも常に黙って削除されていたのを、startTime と同様に onOutOfRange に従わせる（error 時は変換全体を拒否し、hint に範囲外の startTime / pitch を列挙）。
- quantize の最近接丸めがクリップ末尾（`startTime == clipLength`）に到達するとノートが削除されていたのを、範囲内の直前グリッドへ丸めるよう修正する（Live 本体の quantize はノートを削除しない）。strength はこの範囲内ターゲットに向けて補間する。
- onOutOfRange の既定を drop から error に変更し、write_notes の境界検証の既定（拒否）と統一する。削除は `onOutOfRange:"drop"` の明示 opt-in とする。ツール説明・models.yaml に明記。

## 動作確認

- `pnpm test`: pitch 超過の error 拒否（hint に pitch 列挙）・境界値 0/127 の保持・末尾丸め（note@15.9 grid 1 clip 16 → 15、プロパティ保持）・strength 補間・丸めタイ（0.5 → 1）を検証（transform-notes 13 件全緑）。
- `pnpm typecheck`: 全パッケージ成功。
