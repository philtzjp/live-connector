# 2.17.5

schema ツールが申告する select 対象ツール一覧の欠落を修正する patch release。

## 修正内容

- `packages/lom-schema/src/schema.ts` の `query_contract.select.tools` に `insert_device`（#15 導入時の記載漏れ）と `move_clip` / `trim_clip`（#13 導入時の記載漏れ、突合テストで新たに検出）を追加し、トップレベル入力に select を受け取る全 25 ツールと一致させる。
- 一覧と登録ツールの突合テスト（`apps/extension/src/server/select-contract.test.ts`）を追加する。`registerAllTools()`（テスト用に export 化）で実際の登録経路をたどり、`inputSchema` に `select` を持つツール名の集合と `query_contract.select.tools` の完全一致を固定する。batch はステップ内 select のため対象外。
- `llm/models.yaml` の `write_select_return_contract.tools` を同じ 25 ツールに更新する。

## 動作確認

- `pnpm test`: 突合テストが 25 ツールの完全一致で緑（extension 64 件全緑）。
- `pnpm typecheck`: 全パッケージ成功。
